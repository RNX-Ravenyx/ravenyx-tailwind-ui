## ğŸ¨ **Variants**

The `Button` component supports **five types**:

- **Primary** (`type="primary"`) - The main action button
- **Default** (`type="default"`) - A neutral button
- **Dashed** (`type="dashed"`) - A button with a dashed border
- **Text** (`type="text"`) - A button without background
- **Link** (`type="link"`) - Works like an anchor tag
