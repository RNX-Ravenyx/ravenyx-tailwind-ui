# ðŸŽ¨ ThemeProvider

`ThemeProvider` enables **dynamic theming** for the UI library, allowing you to customize colors, fonts, and other UI styles.

---

## ðŸ“Œ **Default Theme**

By default, `ThemeProvider` provides the following values:

```tsx
const defaultTheme = {
  primary: '#1DA1F2',
  secondary: '#14171A',
  background: '#FFFFFF',
  foreground: '#000000',
  fontFamily: 'Inter, sans-serif'
}
```

If no theme is provided, this default theme will be used.

---

# ðŸŽ¨ Customizing the Theme

To override the default theme, wrap your app with `ThemeProvider` and pass a `theme` prop:

```tsx
import { ThemeProvider } from 'tailwind-ui-library'

const customTheme = {
  primary: '#FF5733',
  secondary: '#222831',
  background: '#FAFAFA',
  foreground: '#000000',
  fontFamily: 'Arial, sans-serif'
}

function App() {
  return (
    <ThemeProvider theme={customTheme}>
      <MyComponent />
    </ThemeProvider>
  )
}
```

ðŸŽ­ Example: Using the Theme Inside a Component
Components can access the current theme using the useTheme hook.

```tsx
import { useTheme } from './ThemeProvider'

const ThemedButton = () => {
  const theme = useTheme()
  return <button style={{ backgroundColor: theme.primary, color: theme.foreground }}>Click Me</button>
}
```
